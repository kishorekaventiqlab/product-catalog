## Important Notes

- **Replace `xyz1233`** with the actual productId returned from CREATE operation
- All UPDATE and DELETE operations require an existing productId
- For CREATE operations, productId is auto-generated by the API
- LIST operation returns all products without filters

## Testing Sequence

Recommended order for complete CRUD testing:

1. **CREATE** → Use `create-product.json` → Save returned productId
2. **GET** → Use saved productId to verify creation
3. **LIST** → Verify product appears in list
4. **UPDATE** → Use `update-product.json` or `update-product-partial.json` with saved productId
5. **GET** → Verify updates were applied
6. **DELETE** → Use saved productId to remove product
7. **GET** → Verify product is deleted (should return 404)
